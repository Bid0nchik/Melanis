# Melanis Bot

Telegram бот на основе Groq API для обработки текста, голосовых сообщений и изображений.

## Основные возможности

- **Текстовые сообщения** - ответы от нейросети Groq с поддержкой контекста
- **Анализ изображений** - распознавание и описание фото (llama-3.2-11b-vision)
- **Транскрипция голоса** - преобразование голосовых сообщений в текст (OpenAI Whisper)
- **Автоматическое разбиение** - длинные ответы отправляются несколькими сообщениями
- **Контекст ответа** - бот читает контекст ответа (на какое сообщение вы ответили)
- **Управление чатами** - возможность создавать отдельные чаты с независимым контекстом

## Управление чатами

Бот поддерживает несколько независимых чатов с отдельной историей диалога:

- `/newchat <имя>` - создать новый чат
- `/chats` - показать список чатов и выбрать активный
- `/deletechat <имя>` - удалить чат
- `/clear` - очистить историю текущего чата

Каждый чат имеет свой собственный контекст разговора.

## Запуск

### Локально (для тестирования)
```bash
python main.py
```
Бот автоматически запустится в polling режиме с health check сервером на порту 10000.

### На Render (production)
Бот автоматически использует webhook режим при наличии переменной `RENDER_EXTERNAL_URL`.

## Как начать

### Требования
- Python 3.10+
- Telegram аккаунт

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repo_url>
cd OpenAiStart
```

2. Создайте виртуальное окружение:
```bash
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл `.env` с нужными токенами:
```env
TELEGRAM_BOT_TOKEN=your_telegram_token
GROQ_API_KEY=your_groq_api_key
OPENAI_API_KEY=your_openai_api_key
GROQ_VISION_MODEL=llama-3.2-11b-vision-preview
```

### Запуск локально

```bash
python main.py
```

Бот запустится с polling и будет ждать сообщений.

## Развертывание на Render

1. Подготовьте переменные окружения на Render:
   - `TELEGRAM_BOT_TOKEN`
   - `GROQ_API_KEY`
   - `OPENAI_API_KEY`
   - `GROQ_VISION_MODEL` (опционально)
   - `RENDER_EXTERNAL_URL` (автоматически на Render)

2. Разверните на Render через GitHub

3. Бот автоматически установит webhook и начнет обрабатывать сообщения

## Использование

Просто напишите боту сообщение:
- Текст - получит ответ от нейросети
- Фото - бот проанализирует изображение
- Голос - бот транскрибирует и ответит
- ↩Ответ на чьё-то сообщение - бот будет учитывать контекст

## Используемые API

- **Groq** - LLM для текстовых ответов и анализа изображений
- **OpenAI Whisper** - транскрипция голосовых сообщений
- **Telegram Bot API** - взаимодействие с Telegram

## Архитектура

- **webhook** - для работы на Render (пробуждается при сообщениях)
- **polling** - для локального тестирования (постоянное соединение)
